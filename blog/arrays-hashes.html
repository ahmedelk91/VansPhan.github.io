Hashes VS. Arrays

(۶ૈ ᵒ̌ Дᵒ̌)۶︵ ┻━┻

by Van Phan

  • What is an array?

    An array in an object that consist of many other objects. It has an index for each of its objects starting from 0 and so on.

  • What is a hash?

    A hash is like a library with object that has a key associated with it. There is no particular order in a hash so you must use the key to reference the value you need.

  • When to use an array

    You use an array when the order of the object is important. The index can become very useful to keep a count of the objects.

  • When to use a hash

    Hashes are very useful when you want to make a blueprint for more specific hashes. You could make keys that you want to repeat and then fill out each hash's values differently.
