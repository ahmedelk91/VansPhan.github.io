by Van Phan

  • What is the map method?

    This method invokes the given block once for every one of your item in your array.

  • What does it look like?

    array.map.with_index{ |x, i| x * i } #=> ["", "b", "cc", "ddd"]

  • When can you use it?

    One way you can use the map method is when you need to duplicate an array so that your method can be non-destructive. If you can map an array and make a copy, you can manipulate the copy as much as you like without affecting the original.
